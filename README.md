# vianium/.github
[![License: Apache 2.0](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LICENSE) [![Vianium](https://img.shields.io/badge/org-vianium-00A0F0.svg)](https://github.com/vianium) [![Issues](https://img.shields.io/github/issues/vianium/.github.svg)](https://github.com/vianium/.github/issues)

Organization metadata and shared templates for the
[Vianium](https://github.com/vianium) project.

This repository is special: GitHub uses its `profile/README.md` as the
organization's public profile page. It also hosts default issue and PR
templates and the org-level security policy.

## Contents

| Path | Purpose |
|---|---|
| [profile/README.md](profile/README.md) | Public profile shown at github.com/vianium |
| [.github/ISSUE_TEMPLATE/](.github/ISSUE_TEMPLATE/) | Default issue templates inherited by every repo |
| [.github/PULL_REQUEST_TEMPLATE.md](.github/PULL_REQUEST_TEMPLATE.md) | Default PR template |
| [SECURITY.md](SECURITY.md) | Default org-level security policy |
| [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) | Default org-level code of conduct |
| [CONTRIBUTING.md](CONTRIBUTING.md) | Default org-level contribution guide |

Repository-specific overrides (e.g. a stricter `SECURITY.md` in a single
repo) take precedence over the org-wide defaults stored here.
